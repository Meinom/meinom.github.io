# meinom.github.io

Moniem's blog - built with [Astro](https://astro.build) and the [Minimalistic Astro Blog](https://astro.build/themes/details/minimalistic-astro-blog/) template.

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the blog.

### Build for Production

```bash
npm run build
```

The static site will be generated in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## 📁 Project Structure

```
├── src/
│   ├── pages/          # Page routes
│   ├── styles/         # Global styles
│   └── components/     # Reusable components
├── public/             # Static assets
├── astro.config.mjs    # Astro configuration
└── package.json        # Project dependencies
```

## 📝 Adding Blog Posts

Create `.astro` or `.md` files in the `src/pages/` directory to add new content.

## 🎨 Customization

Edit the styles in `src/styles/global.css` to customize the blog appearance.
